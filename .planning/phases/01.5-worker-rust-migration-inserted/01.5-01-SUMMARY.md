- **Fix:** Changed `canonical_bytes` to return `Result<Vec<u8>, serde_json::Error>`, which is the actual error type from `serde_jcs::to_vec`.
- **Files modified:** `crates/event-schema/src/jcs.rs`
- **Verification:** `cargo test -p event-schema --locked`; `cargo clippy -p event-schema --locked -- -D warnings`
- **Committed in:** `9ad4c5d` and retained in `f44906e`

**2. [Rule 3 - Blocking] Added Cargo.lock updates**
- **Found during:** Task 2 RED
- **Issue:** Plan verification requires `--locked`, but adding `ts-rs` without updating `Cargo.lock` would make locked builds fail.
- **Fix:** Let Cargo resolve and commit the lockfile update with `ts-rs 12.0.1` and transitive packages.
- **Files modified:** `Cargo.lock`
- **Verification:** All `--locked` cargo commands passed.
- **Committed in:** `9ad4c5d`
